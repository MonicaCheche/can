# can
can comunication with zybo7020, stm32g474re, usbtocan(SH-C31A), can bus transceiver (SN65HVD230)
For a Resume or CV (Bullet Points)
## Embedded Firmware & Bus Communication:

Developed and validated real-time CAN/CAN-FD communication interfaces on STM32 microcontrollers, configuring peripheral alternate functions and matching baud rates for reliable bus performance.

Implemented hardware-software integration for automotive networks, combining transceiver wiring, MCU register configuration, and SocketCAN debugging.

## Linux & SoC Integration (Zybo / Zynq):

Configured embedded Linux networking and SocketCAN protocol stacks (can-utils, slcan bridging) on ARM/FPGA heterogeneous SoC platforms to monitor, parse, and verify live bus traffic.
